Enter CUSIP Codes to find out if they're Municipal Search
View added codes All | Municipal Only

API:

Add new codes --
url: /add/
request: POST
parameters: {'csv': ' [csv string of CUSIP Codes] '}

View codes --
url: /get_bonds/:muni/:json
request: GET
parameters: muni [1] (limit to municipal only) [0] (list all)
json [1] (output in JSON format) [0] (output in HTML)
